Тег img

Тег img - это тег который отвечает за добавление картинок на сайт

Тег img одиночный ему не нужен закрывающий тег

Cинтаксис

< img src="путь к изображению">

src — указывает путь к изображению. Может быть относительным или абсолютным

width и height — задают размеры изображения в пикселях.

Их указание помогает браузеру зарезервировать место для изображения заранее, что предотвращает «прыжки» макета при загрузке

Дополнительные атрибуты

srcset — позволяет указать несколько версий изображения с разной шириной для адаптивной загрузки

sizes — определяет, какой размер изображение займёт в вьюпорте при разных условиях

decoding — указывает предпочтительный метод декодирования изображения
