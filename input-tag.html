Тег input

Тег input — это элемент для создания интерактивных элементов форм, которые позволяют пользователю вводить данные

нажимать кнопки и прочее

Cинтаксис

Тег input — одиночный то есть у него нет тега закрывающего тега как у парных

Пример: < input type="text" name="username">

Основные типы полей

text — однострочное текстовое поле

checkbox — флажки для выбора нескольких вариантов

radio — круговые кнопки-переключатели для выбора одного из нескольких вариантов

submit — кнопка для отправки формы

button — кнопка

color — виджет для выбора цвета

file — выбор файла для отправки

Некоторые aтрибуты

placeholder — предоставляет подсказку в поле ввода

readonly — делает поле доступным только для чтения

pattern — задаёт регулярное выражение для проверки значения поля
