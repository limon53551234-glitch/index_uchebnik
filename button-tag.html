Тег button

Тег button - это тег с помощью которого можно создавать кнопки

Он позволяет запускать действия:

отправку данных формы

запуск JavaScript-функций и тд

Синтаксис

Тег button парный: то есть < button> < /button>

Внутрь тега < button> можно помещать текст, и другие HTML-элементы.

Внешний вид кнопки можно изменять с помощью CSS.

Атрибуты

К тегу button можно применить универсальные атрибуты

form — связывает кнопку с формой через её ID

type — задаёт тип кнопки

и тд

P.S

В тег button можно поставить не только текст но и картинки!
