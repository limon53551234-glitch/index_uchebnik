Тег P

Тег p— это HTML-тег, который определяет текстовый абзац.

Он используется для написания текстового контента на веб-странице, например, в статьях, блогах, описаниях товаров.

Основные характеристики

Автоматическое добавление отступов.

Браузеры по умолчанию добавляют пробелы (отступы) до и после абзаца, визуально отделяя один текстовый

блок от другого. Величиной отступов можно управлять с помощью CSS.

Синтаксис

Для тега p доступны несколько атрибутов, которые применимы ко всем HTML-элементам. Примеры:

id — уникальный идентификатор абзаца, используется для CSS-стилизации и создания якорных ссылок

class — назначает абзацу один или несколько классов для применения стилей

title — добавляет всплывающую подсказку при наведении курсора

и тд

P.S

С помощью тега P можно разместить не только обычный текст но и ссылки
